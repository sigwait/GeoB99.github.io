<h2 id="user32-user-mode-library-layout-tree">User32 user-mode library layout tree</h2> <p><strong>User32.dll</strong> is a user mode DLL library that provides windowing manager APIs to the client. An example of a call brought by the said library is <code>CreateWindowExW</code> which creates a window with extended styles. The documented function calls in the <strong>User32.dll</strong> library rely on undocumented internal functions in Win32k, the kernel mode side of the Windows subsystem.</p> <ul class = "tree">
